Welcome to the team. This page covers the accounts and tools you need to get set up.

## Accounts to request

Ask your manager or a teammate to invite you to:

| Tool | What for |
|---|---|
| WordPress admin | Editing the site content |
| GitHub (org member) | Accessing the codebase and this docs site |
| Cloudflare | DNS and CDN access (technical roles only) |
| SiteGround | Hosting dashboard (technical roles only) |

## WordPress access levels

| Role | Can do |
|---|---|
| Editor | Create, edit, publish any content |
| Author | Create and publish their own content only |
| Contributor | Write drafts, cannot publish |

Your manager will assign the right role when they create your account.

## First things to read

- [Content editing](/team/global/content-editing) — how to update pages and posts
- [Media guidelines](/team/global/media) — image sizes, formats, file naming
- [Publishing workflow](/team/global/publishing) — how content gets reviewed and published

## Who to contact

For access issues, contact your manager.
For site bugs or anything broken, post in the team Slack channel.
