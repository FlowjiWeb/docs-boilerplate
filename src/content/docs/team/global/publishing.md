## Who can publish

| Role | Can publish |
|---|---|
| Editor | Yes — any content |
| Author | Yes — their own content only |
| Contributor | No — must submit for review |

## Standard workflow

```
Draft → Review → Publish
```

1. **Write** your content and save as draft
2. **Preview** it using the Preview button — check for typos, layout issues, broken links
3. **Submit for review** (Contributors) or publish directly (Authors, Editors)
4. If submitted for review, an Editor will check and either publish or leave a comment

## Time-sensitive content

For content that must go live at a specific time (announcements, campaigns):

1. Write and get it reviewed in advance
2. In the editor, click **Publish → Schedule**
3. Set the date and time, click **Schedule**

WordPress will publish it automatically at the scheduled time.

## Unpublishing content

To take a page or post offline without deleting it:

1. Open the page editor
2. In the right sidebar, change **Status** from **Published** to **Draft**
3. Click **Update**

The page is now offline but preserved — you can republish at any time.

## Before you publish — checklist

- [ ] Spelling and grammar checked
- [ ] All links work (click them)
- [ ] Images have alt text
- [ ] Meta description filled in (SEO field below the editor)
- [ ] Previewed on mobile (use the **Preview → Mobile** option in the editor toolbar)
