## Pages

Standard WordPress pages. Used for all permanent sections of the site — homepage, about, services, contact, etc.

Only Editors can publish pages. If you need a new page created, ask an Editor or raise a request in Slack.

## Posts (Blog)

Regular blog articles. Authors can write and publish their own posts without Editor approval.

**Required fields before publishing:**
- Featured image (minimum 1200 × 630 px)
- Excerpt (2–3 sentences, shown in listing pages)
- Category (at least one)
- Yoast SEO fields (see [SEO guidelines](/team/website-1/seo))

## Case studies

Custom post type. Each case study has structured fields:

| Field | Description |
|---|---|
| Client name | Displayed in the header |
| Industry | Used for filtering |
| Challenge | Short paragraph, the client's problem |
| Solution | What was built or done |
| Result | Outcome, ideally with a metric |
| Testimonial | Optional quote from the client |

## Team members

Custom post type for the team page. Fields: name, role, bio (short), photo, LinkedIn URL.

## Press / media

Custom post type for press mentions. Fields: publication name, article title, date, URL, logo.
