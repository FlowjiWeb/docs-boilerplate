title: SEO guidelines
description: How to fill in SEO fields on Website 1
---

Every page and post has a **Yoast SEO** panel below the editor. Fill it in before publishing.

## SEO title

The title shown in Google search results. Keep it under 60 characters.

- Good: `Custom WordPress Development — ENV_NAME`
- Bad: `Home` or a title that's 90 characters long

If you leave this blank, Google will use the page title — which is usually fine for posts but too generic for pages.

## Meta description

The snippet shown under the title in search results. 120–155 characters, written to make someone want to click.

- Include the main keyword naturally
- Don't stuff keywords — write for the reader
- Every page should have a unique meta description

## Focus keyword

The main search term this page should rank for. Yoast uses this to score the content — aim for green.

## Open Graph (social sharing)

When the page is shared on social media, it uses the **Social** tab in Yoast. If you leave it blank, it falls back to the SEO title and meta description.

For important pages and any content you expect to be shared, set a dedicated OG image (1200 × 630 px).

## Checklist before publishing

- [ ] SEO title set and under 60 characters
- [ ] Meta description set, 120–155 characters
- [ ] Focus keyword set
- [ ] Yoast traffic light is green or orange (red = fix before publishing)
- [ ] Featured image set (also used as OG image fallback)
