## Adding a new product

1. **Products → Add New** in the WP admin
2. Set the product name and write a description in the main editor
3. In the **Product data** panel, select the product type (Simple, Variable, etc.)
4. Set the price
5. Add the product to a **Category** and add **Tags** if relevant
6. Upload a **Product image** (main image) and any **Product gallery** images
7. Fill in **Yoast SEO** fields
8. Publish

## Product types

| Type | When to use |
|---|---|
| Simple | Single item, one price |
| Variable | Multiple options (size, colour, format) |
| Grouped | Collection of related products shown together |
| External | Links out to another site for purchase |

## Stock management

Stock is tracked per-product. To update stock:

1. Open the product
2. **Product data → Inventory**
3. Check **Manage stock** and enter the current quantity

When stock hits zero, the product automatically shows as "Out of stock" and disables the add-to-cart button.

## Pricing and sales

To put a product on sale:
1. Open the product
2. **Product data → General**
3. Enter a **Sale price** and optionally set a **Schedule** (start and end date)

The original price will show struck through automatically.
